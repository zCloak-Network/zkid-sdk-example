## Quick Use

Make sure using the latest version of SDK: `npm update`

Recommended node version: **v18.16.1 LTS**

```bash
git clone https://github.com/zCloak-Network/zkid-sdk-example.git
cd zkid-sdk-example/
npm install

npm run ctype
npm run claim
npm run attest
npm run issue
npm run vp-send
npm run vp-send-verify
npm run zkp
```
